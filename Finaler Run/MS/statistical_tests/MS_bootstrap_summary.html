Interpretation

The bootstrap analysis assesses whether the trading strategy's performance is statistically different from what would be expected by chance. A p-value less than 0.05 generally indicates statistical significance, suggesting the strategy's returns are unlikely to be due to random chance.

The 95% confidence interval represents the range within which the true mean return is likely to fall with 95% probability based on the bootstrap simulations. If this interval does not include zero, it suggests statistical significance.
